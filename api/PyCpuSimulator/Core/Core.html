class PyCpuSimulator.Core.Core.StandardCore(memories)[source]¶

Bases: PyCpuSimulator.Core.Core.Core

This class implements a standard core with a very felxible ALU. For example we can add directly two constants.

ALU operations are performed using Python integer. The value is truncated to the data type in the Numpy cell.
